Keyboard shortcuts

We've added a lot of useful shortcuts to the translation dialog box, translation pop-ups, and so on, and now you can use the keyboard for most of the：

  • Translate dialog shortcuts：

    • Displays a list of source - Alt S
    • Displays a list of target languages - Alt T
    • Exchange languages - Alt , Shift , S
    • Switch window fixed state - Alt P
    • Play TTS - Alt/Meta/Shift Enter
    • Favorite To Word Book - Ctrl/Meta F
    • Show History - Ctrl/Meta - H
    • Copy translation - Ctrl/meta Shift C
    • Empty input - Ctrl/Meta Shift BackSpace/Delete
    • Expand more translations - Ctrl/Meta and Down
    • Put away more translations - Ctrl/Meta UP

  • Translate bubble shortcuts：

    • Open with a dialog box - Ctrl , Shift , Y / Control , Meta , U

  • Quick document window shortcuts：

    • Turn automatic translation on/off - Ctrl , Shift , Y / Control , Meta U

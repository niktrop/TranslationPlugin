マイクロスケール

プラグイン用に新しいマイクロスカラーを設計：

翻訳

[翻訳] ダイアログ ボックス

翻訳ダイアログのユーザー インターフェイスが再設計され、翻訳ダイアログがより簡潔で使いやすされました。

キーボード ショートカット

翻訳ダイアログ、翻訳ポップアップなどのインターフェイスに実用的なショートカットキーをたくさん追加し、キーボードを使用してほとんどの操作を実行：

  • 翻訳ダイアログのショートカット：

    • ソース言語の一覧を表示 - Alt + S
    • ターゲット言語の一覧を表示 - Alt + T
    • 言語を入れ替 - Alt + Shift + S
    • ウィンドウの固定状態を切り替える - Alt + P
    • TTS を再生 - Alt/Meta/Shift キーを押しながら Enter を します
    • 単語帳にお気に入り - Ctrl/メタ + F
    • 履歴の表示 - Ctrl/Meta + H
    • 翻訳のコピー - Ctrl/Meta + Shift + C
    • 入力を空にする - Ctrl/Meta + Shift + BackSpace/Delete を空にします
    • その他の翻訳を展開する - Ctrl/Meta を + Down を
    • より多くの翻訳を片付け - Ctrl/Meta + UP

  • バブルショートカットの翻訳：

    • ダイアログボックスで開く - Ctrl + Shift + Y / Control + メタ + U

  • クイック ドキュメント ウィンドウのショートカット：

    • 自動翻訳のオン/オフ - Ctrl + Shift + Y / Control + メタ + U

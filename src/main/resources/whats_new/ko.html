Translation 3.0의 새로운 기능

Translation v3.0을 사용해주셔서 감사합니다. 해당 버전에는 다음과 같은 주요 업데이트가 포함되어 있습니다:

  • 새로운 플러그인 로고 - 플러그인 로고를 새롭게 디자인 했습니다.
  • 새로운 사용자 인터페이스 - 새로운 번역 대화 상자 및 설정 페이지 UI
  • 더 많아진 키보드 단축키 - 아직 마우스를 사용하시나요?이제 없어도 됩니다!
  • 디스크 캐시 - 디스크 캐시 지원

번역 대화상자

번역 대화 상자의 사용자 인터페이스가 재설계되어 번역 대화 상자를 보다 간결하고 쉽게 사용할 수 있습니다.

키보드 단축키

번역 대화 상자, 번역 팝업 창 등의 인터페이스에 여러 키보드 단축키가 추가되어, 이제 키보드 단축키를 사용하여 대부분의 작업을 할 수 있게 되었습니다.：

  • 번역 대화 상자 단축키:

    • 소스 언어 목록 표시 - Alt + S
    • 대상 언어 목록 표시 - Alt + T
    • 언어 전환 - Alt + Shift + S
    • 창을 고정/고정 해제 - Alt + P
    • TTS 재생 - Alt/Meta/Shift + Enter
    • 단어장에 저장 - Ctrl/Meta + F
    • 기록 표시 - Ctrl/Meta + H
    • 번역 복사 - Ctrl/Meta + shift + C
    • 입력 내용 지우기 - Ctrl/Meta + Shift + BackSpace/Delete
    • 더 많은 번역 펼치기 - Ctrl/Meta + Down
    • 더 많은 번역 수신 - Ctrl/Meta + UP

  • 번역 말풍선 단축키:

    • 대화 상자로 열기 - Ctrl + Shift + Y / Control + Meta + U

  • 빠른 문서 창 단축키:

    • 자동 번역 켜기/끄기 - Ctrl + Shift + Y / Control + Meta + U

빠른 문서 번역

콘텐츠 길이가 번역 제한을 초과하는 문서의 경우 번역이 건너뛰고 프롬프트가 표시됩니다：

디스크 캐시입니다

이제 번역 결과가 디스크에 캐시되고 다음 번에 동일한 번역이 캐시에서 직접 읽습니다.물론 설정 페이지에서 캐시를 정리할 수 있습니다：

페이지를 설정합니다

설정 페이지의 사용자 인터페이스를 다시 배치했으며 이제 IDEA의 UI 스타일에 더 잘 부합합니다.또한 설정 페이지가 이제 설정도구 > 그룹으로 이동되었습니다.아, 찾을 수 없다는 것에 대해 걱정할 필요가 없으며 번역 대화 상자에 눈에 띄는 위치에 포털을 배치했으며 여기에서 설정 페이지를 빠르게 열 수 있습니다.
